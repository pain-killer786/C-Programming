A function is a block of code that performs a particular task.

Features of Function:-
    It can be used multiple times.
    It increases code reusability.

Properties of Function:-
    Execution always starts from main
    A  function gets called directly or indirectly from main
    There can be multiple functions in a program.

Argument
    Values that are passed in a function call
    used to send Values
    Actual Parameter

Parameter
    Values in function declaration and defination
    Used to receive Values
    Formal Parameter

Note
    Function Can only return one value at a time 
    Changes to parameters in function don't change the values in calling function 

Recursion
    When a function calls itself
